Read the markdown file at the path provided. Find the `## Implementation Plan` section and carefully read every step.

Then execute the implementation plan step by step:

- Work through each numbered step in order
- Use the appropriate tools to make actual code changes (Edit, Write, Bash, etc.)
- After completing each step, briefly confirm what was done before moving to the next
- If a step is ambiguous, make a reasonable assumption and note it
- Do not skip steps — complete all of them

When finished, summarize what was implemented.

The markdown file to read: $ARGUMENTS
